🌊 Itinerario de Fin de Semana en Gijón

Un sitio web interactivo y estético diseñado para presentar un plan de viaje personalizado para un fin de semana en Gijón, Asturias. El proyecto combina un diseño moderno ("aesthetic") con información práctica para los viajeros.

✨ Características

Diseño Responsivo: Adaptado para móviles, tablets y ordenadores.

Estética Premium: Uso de tipografías elegantes (Playfair Display y Montserrat) y una paleta de colores inspirada en el mar Cantábrico (Azul Marino y Beige).

Mapas Interactivos: Integración de Google Maps para localizar restaurantes y puntos de interés.

Sección Especial de Siesta: Un bloque diseñado específicamente para el descanso, optimizando la experiencia del usuario.

Función de Impresión: Botón personalizado para guardar el itinerario como PDF o imprimirlo físicamente.

Fallback de Imágenes: Gestión de errores para cargar imágenes alternativas si las originales no están disponibles.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica.

Tailwind CSS: Framework de estilos utilizado vía CDN para un desarrollo ágil y moderno.

Google Fonts: Tipografía de alta calidad.

JavaScript: Lógica para la funcionalidad de impresión y configuraciones dinámicas de Tailwind.

📍 El Plan de Viaje

El itinerario cubre 4 días de exploración:

Día 1: Bienvenida, paseo por Cimavilla y cena en "El Chigre 2.0".

Día 2: Excursión a Cudillero, incluyendo el "Secret Passage" y los miradores panorámicos.

Día 3: Visita a la Universidad Laboral y tarde de vermut.

Día 4: Despedida relajada en el Puerto Deportivo.

🚀 Cómo Visualizarlo

Para ver el proyecto localmente:

Clona este repositorio o descarga el archivo itinerario_gijon.html.

Abre el archivo .html en cualquier navegador web moderno (Chrome, Safari, Firefox, Edge).

No requiere instalación de dependencias externas ya que utiliza CDNs para los estilos.
